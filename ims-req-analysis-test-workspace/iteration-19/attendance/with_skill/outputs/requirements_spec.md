## 1. Overview + Business Context

考勤模块是IMS人事(HR)模块下的核心子模块，支撑制造企业员工考勤全生命周期管理。覆盖考勤数据采集（考勤机同步/导入/手工/补卡）、日考勤统计计算、月考勤汇总、以及多维度报表。

**关键特征：**
- 多数据源：考勤机同步、TXT/Excel导入、手工补卡、系统自动生成
- 复杂班次计算：早班/午班/晚班三班制，支持孕期/哺乳期差异化时间
- 假类体系：年假、病假、事假、特殊假 + 6个可配置拓展假类
- 版本管理：月考勤统计支持多版本，旧版本作废后可追溯

**数据流向：**
```
考勤机/TXT/Excel/补卡 → 考勤登记 → 日考勤统计 → 月考勤统计
回工登记(生效) → 生成回工考勤记录 → 日考勤统计
请假登记(生效) → 生成日请假明细 → 日考勤统计
日考勤统计 → 小组出勤表 / 员工缺勤表 / 月考勤明细表
```

## 2. Functional Points

### F001 考勤登记 - CRUD与数据采集
考勤记录新增（考勤机同步、Excel导入、TXT导入、手工补卡）、查询、删除、导出

### F002 考勤登记 - 数据校验
工号关联员工档案、考勤日期范围限制3个月、来源为系统的记录不可删除

### F003 补卡功能
手工补卡：补卡类型（正常/出差）、考勤日期+时间、来源标记为补卡、自动触发日考勤统计

### F004 回工登记 - 生命周期管理
新增、编辑（仅草稿）、删除（仅草稿）、生效、反生效

### F005 回工登记 - 回工考勤记录生成
根据班次规则计算早班/午班回工工时，生成对应考勤记录（来源=系统），自动计算日考勤统计

### F006 请假登记 - 5态生命周期
草稿 → 审批中 → 生效 / 关闭 / 取消；编辑/删除仅草稿；生效/反生效；自动生成日请假明细

### F007 请假登记 - 日请假生成
按请假时间范围生成日请假明细，排除周末/节假日，计算早班/午班请假工时，请假自动删除签到时间（可配置）

### F008 日考勤统计 - 计算引擎
增量插入在职员工、获取班次设置、更新日期类型（工作日/周末/节假日）、更新孕期/哺乳期签到时间、更新各班上下班时间、计算各班工作时数/迟到/早退、更新回工/假期工时、两次统计汇总

### F009 日考勤统计 - 快速补卡
早班/午班上下班快速补卡，根据已有打卡状态和请假状态决定按钮启用/禁用，支持默认标准时间补卡或自定义补卡

### F010 日考勤统计 - 批量补卡
选择多条日考勤记录批量补卡，校验日期/班次一致性，支持覆盖已打卡/覆盖请假选项

### F011 日考勤统计 - 更改班次
批量更改选中记录的班次，重算日考勤

### F012 日考勤统计 - 日考勤计算
按条件重新计算日考勤，日期范围限制3个月，显示进度条

### F013 日考勤统计 - 明细查看
查看具体考勤记录（出勤/请假），支持删除考勤记录（来源非系统）、请假反生效/生效

### F014 日考勤统计 - 排序
多字段自定义排序，保存本地

### F015 日考勤统计 - 考勤同步
从考勤机Access数据库同步数据，记录同步截止时间

### F016 月考勤统计 - 生成与版本管理
按月汇总日考勤统计，旧版本作废，新版本号+1，支持重新计算

### F017 月考勤统计 - 详情查看
版本下拉切换、明细数据展示、导出、员工工资导出

### F018 月考勤明细表
按员工维度打印月考勤明细，支持批量打印（上限500人），过滤假类

### F019 小组出勤表
按日期/部门/组别汇总出勤人数、缺勤人数，近3月数据

### F020 员工缺勤表
按工号汇总缺勤天数>0的员工，近3月数据

## 3. Business Rules

### R001 考勤日期查询约束
考勤登记/日考勤统计/小组出勤表/员工缺勤表：查询跨度≤3个月；请假登记：查询跨度≤1年

### R002 删除约束
来源=系统的考勤记录不可删除（通过回工登记/请假登记自动生成的）

### R003 补卡/导入后自动计算
补卡、导入完成后自动对相关员工及日期生成/更新日考勤统计

### R004 回工登记生效条件
员工数必须>0，否则告警"没有登记员工，不能生效。"

### R005 回工登记反生效约束
回工日期<今天时，confirm("回工日期小于今日，反生效会影响已有考勤统计，是否继续？")

### R006 请假登记冲突校验
[开始日期+开始时间]到[结束日期+结束时间]范围内不能存在该员工的请假登记

### R007 请假登记反生效约束
开始时间<今日时，confirm("开始时间小于今日，反生效会影响已有考勤统计，是否继续？")

### R008 日考勤计算日期约束
起始/结束日期不可超出3个月前或大于今天

### R009 月考勤明细表人数约束
员工人数超过500时，提醒"员工已经超出500人。"并停止

### R010 批量补卡约束
选中记录必须同日期、同班次

### R011 快速补卡按钮禁用逻辑
- 早班上班补卡：早班上班时间>0且≤标准早班上班时间，或存在早班请假时数>0
- 早班下班补卡：早班下班时间>0且≥标准早班下班时间，或存在早班请假时数>0
- 午班上班补卡：午班上班时间>0且≤标准午班上班时间，或存在午班请假时数>0
- 午班下班补卡：存在午班请假时数>0

### R012 0.5小时四舍五入规则
加班时数按0.5小时四舍五入：如1.149H→1H、1.15H→1.5H、1.649H→1.5H、1.85H→2H

### R013 回工登记保存自动生效
参数设置：保存回自动生效为真时，保存后自动触发生效

### R014 请假登记保存自动生效
保存后状态为草稿，并自动生效

### R015 请假自动删除签到（可配置）
系统配置项"人事-请假自动删除签到时间"为是时，请假生效后删除对应班次的签到考勤记录

### R016 班次获取优先级
员工档案直接设置班次 > 部门ID+日期获取最新班次

### R017 考勤异常判断
- 没上班：工作日且早/午班上下班时间全空且未请假
- 缺卡：工作日且非"没上班"，但存在早班或午班上下班时间为空且未请假
- 正常：有打卡或已请假覆盖
- 周末/节假日：始终正常

### R018 考勤起扣时长
系统配置"人事-考勤起扣时长"，迟到早退超此值才扣除

### R019 工时计算公式
- 工作日：正班时数 = 早班时数+午班时数-迟到早退扣除时长（独立考核时）
- 工作日加班：晚班时数-晚班迟到早退（独立考核时）
- 周末加班：早班+午班+晚班时数-迟到早退扣除
- 节假日加班：同周末
- 出勤时数 = 正班时数+带薪假时数

## 5. Cross-Module Dependencies

|---------|---------|------|
| 人事-员工档案 | 工号、姓名、部门、组别、班次、设备登录号、在职/离职状态、孕期/哺乳期 | 考勤登记员工匹配、日考勤员工增量、班次获取 |

## 6. Data Entities & Variations

### 6.1 考勤记录
| 字段 | 类型 | 约束 | 来源 |
|------|------|------|------|
| 工号 | 文本(20) | 关联员工档案 | 导入/同步/手工 |
| 考勤时间 | 日期时间(到分钟) | 来源决定 | 各来源 |
| 类型 | 枚举 | 正常/出差/回工/病假/事假/特殊假/年假+6拓展 | 业务决定 |
| 来源 | 枚举 | 考勤机/补卡/导入/系统 | 操作决定 |

### 6.2 考勤类型变体
| 类型 | 来源 | 触发方式 |
|------|------|---------|
| 正常 | 考勤机/导入/补卡 | 考勤同步/文件导入/补卡操作 |
| 回工 | 系统 | 回工登记生效 |
| 各类假 | 系统 | 请假登记生效 |

### 6.3 日考勤统计时间维度
- 早班：上班签到/下班签退
- 午班：上班签到/下班签退
- 晚班：上班签到/下班签退（加班核心）

### 6.4 特殊人群维度
| 人群 | 差异化字段 |
|------|----------|
| 正常 | 标准上班/下班时间 |
| 孕期 | 孕期上班/下班时间（推迟上班、提前下班） |
| 哺乳期 | 哺乳期上班/下班时间 |

## 7. Edge Cases & Constraints

### 7.1 边界约束
| 约束 | 详情 |
|------|------|
| 考勤日期查询 | 最大跨度3个月 |
| 请假日期查询 | 最大跨度1年 |
| 日考勤计算日期 | 不能超出3个月前，不能大于今天 |
| 月考勤明细表 | 员工人数上限500人 |
| 年查询 | 只能查近3年 |
| 工号长度 | 20字符 |
| 事由长度 | 200字符 |
| 考勤机名称长度 | 20字符 |
| 同步间隔分钟数 | 默认10分钟 |
| 小组出勤表 | 最多400个小组 |

### 7.2 状态边界
| 实体 | 状态 | 边界规则 |
|------|------|---------|
| 回工登记 | 草稿/生效 | 仅草稿可编辑/删除；仅生效可反生效 |
| 请假登记 | 草稿/审批中/生效/关闭/取消 | 仅草稿可编辑/删除；生效↔草稿(反生效) |
| 月考勤统计 | 生效/作废 | 生成新版本时旧版本作废 |

### 7.3 异常场景
| 场景 | 处理 |
|------|------|
| 工号不存在 | 提醒"工号不存在。"并停止 |
| 补卡员工数=0 | 告警"没有登记员工，不能生效。" |
| 批量删除未选择 | 提醒"没有选择记录" |
| 补卡未选日期 | 告警"请选择记录。" |
| 补卡日期不同 | 告警"请选择相同日期。" |
| 补卡班次不同 | 告警"请选择相同班次。" |
| 补卡未选时段 | 提醒"请选择考勤时段。" |
| 补卡无时间 | 提醒"请输入选中考勤时段的补卡时间。" |
| 请假时间重叠 | 提醒"请假时间范围内存在其他请假。" |
| 员工超500 | 提醒"员工已经超出500人。" |
| 日考勤计算日期非法 | 提醒"起始/结束日期不可超出3个月前或大于今天。" |

### 7.4 工时计算边界
| 场景 | 处理 |
|------|------|
| 未打卡 | 对应班次时数=0 |
| 工作时长<起计时长 | 该班次时数=0 |
| 迟到<免计时长 | 迟到=0 |
| 早退<免计时长 | 早退=0 |
| 午班晚班连续自动考勤 | 午班有签到→晚班自动取标准上班时间 |
