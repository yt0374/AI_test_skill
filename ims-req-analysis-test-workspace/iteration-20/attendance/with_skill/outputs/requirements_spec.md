## 1. Overview + Business Context

考勤模块是IMS智能制造协同平台人事子模块的核心组成部分，负责员工考勤数据的全生命周期管理：从考勤数据采集（考勤机同步/导入/补卡），到日考勤统计计算，再到月考勤统计汇总，最终输出工资计算所需的月考勤数据。该模块还包含回工登记、请假登记、小组出勤表、员工缺勤表等辅助功能。

### 1.1 业务背景

- 服装制造行业，员工数量大（100xxx-108xxx, 8000+ employees）
- 班次制管理：早班/午班/晚班，支持一班制/二班制/三班制
- 考勤机数据同步（远程Access库）+ Excel/TXT导入 + 补卡
- 特殊人群管理：孕期/哺乳期员工有差异化考勤规则
- 假类体系：年假/病假/事假/特殊假 + 6个可配置拓展假类
- 回工机制：工人借调/加班的回工登记与工时计算
- 月末汇总生成工资项输入

### 1.2 模块组织结构

```
人事 (gm.人事#1)
├── 考勤登记          # 考勤机同步、导入、手工、补卡、删除

├── 请假登记          # 草稿→生效→关闭/取消 状态机 + 日请假明细生成
├── 日考勤统计        # 日汇总计算（核心计算引擎）
│   ├── 考勤同步      # 考勤机数据库直接读取
│   ├── txt/Excel导入  # 文件导入
│   ├── 批量补卡      # 故障批量补卡
│   ├── 更改班次      # 临时借调
│   ├── 日考勤计算    # 重算引擎
│   ├── 快速补卡      # 早/午/晚班上下班快速补卡（4个按钮）
│   ├── 请假          # 快速进入请假登记
│   ├── 明细          # 出勤记录 + 请假记录
│   ├── 排序          # 多字段排序本地保存
│   └── 报表          # 月考勤明细表/导出
├── 月考勤统计        # 月汇总 + 版本管理
│   ├── 生成月考勤    # 调用月统计存储过程
│   ├── 详情          # 版本查看 + 重新计算
│   └── 报表          # 月考勤明细表/员工工资/导出
├── 小组出勤表        # 按日/部门/组别汇总
└── 员工缺勤表        # 缺勤员工追踪
```

## 2. Functional Points

### F001: 考勤登记 - 考勤记录管理
- 考勤机同步（Access远程库 → IMS）
- Excel导入 / TXT导入
- 手工补卡（单条）
- 删除（来源非系统才可删，confirm二次确认）
- 批量删除（来源非系统才可删）
- 导出Excel

### F002: 考勤登记 - 查询
- 日期范围（≤3月跨度，快速查今日/本周/本月/本年/昨日/上周/上月）
- 组别/员工/类型/部门/来源 多条件筛选

### F003: 补卡
- 单条补卡（工号模糊匹配、补卡类型正常/出差、日期+时间）
- 非空校验、工号存在校验
- 继续添加功能

### F004: 回工登记 - CRUD + 状态机
- 新增/编辑/删除（仅草稿状态）
- 生效（草稿→生效，需有员工数>0）
- 反生效（生效→草稿，confirm处理）
- 回工考勤记录自动生成（复杂工时计算逻辑）

### F005: 回工登记 - 编辑界面
- 回工日期/时长/时间/部门/组别/事由 填写
- 员工选择界面（6x10网格，全选/全不选）
- 回工时长联动（全天/上午/下午 → 自动设置开始/结束时间）
- 保存后自动生效配置项

### F006: 请假登记 - 状态机
- 草稿→生效（生成日请假明细）
- 反生效（生效→草稿，有日期判断+confirm）
- 编辑/删除（仅草稿状态）
- 日请假生成（复杂时间计算 + 删除签到配置项）

### F007: 请假登记 - 编辑界面
- 员工/假类/请假时长/日期/时间/事由/证明
- 请假时长联动（全天/上午/下午）
- 时段重叠校验
- 保存并新增功能
- 上传证明（多附件）

### F008: 日考勤统计 - 核心计算引擎
- 逐天生成日考勤统计
- 班次规则匹配（员工班次 > 部门班次）
- 早/午/晚班上下班时间计算
- 迟到/早退计算（含孕期/哺乳期特殊规则）
- 工作时数计算
- 回工/假期工时汇总
- 加班时数计算（按0.5小时四舍五入）
- 考勤异常判定（正常/没上班/缺卡）
- 日期类型标记（工作日/周末/节假日）

### F009: 日考勤统计 - 查询
- 日期范围（≤3月跨度）
- 员工/组别/考勤异常/部门/人事状态/孕期/哺乳期/迟到早退/日期类型 筛选

### F010: 日考勤统计 - 操作
- 考勤同步 / TXT导入 / Excel导入
- 批量补卡 / 更改班次 / 日考勤计算
- 快速补卡（早班上下班/午班上下班，含禁用逻辑）
- 请假登记 / 明细查看
- 批量删除 / 排序 / 报表

### F011: 月考勤统计 - 月汇总
- 生成月考勤（按月汇总日考勤）
- 版本管理（作废旧版+1）
- 详情查看 + 版本切换
- 重新计算

### F012: 小组出勤表
- 按日期/部门/组别汇总
- 出勤人数/缺勤人数计算
- 导出/打印

### F013: 员工缺勤表
- 缺勤员工追踪（日标准工时-正班时数-请假时数>0）
- 应出天数/正班天数/请假天数/缺勤天数
- 查询近3月数据

## 3. Business Rules

### R001: 考勤记录删除规则
- 来源=系统 的考勤记录不可删除
- 删除前confirm('是否确认删除？')
- 删除后自动重新计算日考勤统计

### R002: 查询日期跨度规则
- 考勤登记/日考勤统计/小组出勤/员工缺勤: 最大3个月跨度
- 请假登记: 最大1年跨度
- 月考勤统计: 最大3年跨度

### R003: 补卡工号校验
- 从员工档案校验工号存在性
- 工号不存在 → '工号不存在。' 并停止

### R004: 回工登记状态机
- 仅草稿可编辑/删除
- 仅草稿可生效（需员工数>0）
- 仅生效可反生效
- 反生效前confirm处理

### R005: 请假登记状态机
- 草稿→生效→日请假生成
- 反生效：日期<今日需confirm
- 仅草稿可编辑/删除

### R006: 日考勤统计生成规则（详见 §3 日考勤统计生成）

### R007: 小时数四舍五入规则
- 工作日/周末/节假日加班时数: 按0.5小时四舍五入
- 例: 1.149→1H, 1.15→1.5H, 1.649→1.5H, 1.85→2H

### R008: 孕期/哺乳期特殊规则
- 孕期/哺乳期员工有不同的上班/下班时间标准
- 签到/签退窗口偏移计算
- 迟到/早退免计时长仍适用

### R009: 考勤异常判定
- 工作日: 没上班/缺卡/正常
- 周末/节假日: 一律正常

### R010: 日期显示样式
- 周末: 绿色字体
- 节假日: 红色字体
- 考勤异常: 工号红色字体

### R011: 批量补卡前置校验
- 必须选择记录（否则'请选择记录。'）
- 日期必须相同（否则'请选择相同日期。'）
- 班次必须相同（否则'请选择相同班次。'）

### R012: 快速补卡按钮禁用规则
- 早班上班补卡: (早班上班时间>0 AND <=标准) OR 早班请假时数>0 → 禁用
- 早班下班补卡: (早班下班时间>0 AND >=标准) OR 早班请假时数>0 → 禁用
- 午班上班补卡: (午班上班时间>0 AND <=标准) OR 午班请假时数>0 → 禁用
- 午班下班补卡: 午班请假时数>0 → 禁用

### R013: 月考勤统计导出
- 出勤天数<标准上班天数 → 红色异常标记

### R014: 日考勤计算约束
- 起始日期/结束日期不可超出3个月前或大于今天
- 运算量大，需显示进度条

### R015: 月考勤明细表约束
- 员工数>500 → '员工已经超出500人。'并停止

## 4. Data Flow

```
考勤数据采集
├── 考勤机(Access DB) ──同步──→ 考勤登记表
├── TXT/Excel 文件 ──导入──→ 考勤登记表
├── 补卡(手工) ────录入──→ 考勤登记表
└── 回工生效 ──自动生成──→ 考勤登记表(类型=回工,来源=系统)

                ↓
        日考勤统计计算引擎
        (考勤登记 + 员工档案 + 班次设置 + 节假日设置)
                ↓
        日考勤统计表 (每日每位员工一行)
        ├── 早/午/晚班上下班时间
        ├── 迟到/早退(分班次)
        ├── 工作时数/正班时数/加班时数
        ├── 各类假期时数
        ├── 回工时数
        └── 考勤异常标记

请假登记(生效)
    ↓
日请假明细生成
    ↓
更新日考勤统计(请假时数 + 可选删除签到)

                ↓
        月考勤统计生成
        (汇总日考勤统计)
                ↓
        月考勤统计表(每月每位员工一行)
        ├── 各类天数(正班/假期/加班/回工)
        ├── 加班次数
        ├── 迟到早退汇总
        └── 版本管理
                ↓
        员工工资导出
```

## 5. Data Entities & Variations

### 员工（Employee）
| 维度 | 变体 | 影响功能 |
|------|------|---------|
| 人事状态 | 在职/离职 | 离职仅近3月可见 |
| 孕期 | 是/否 | 签到/退窗口偏移 |
| 哺乳期 | 是/否 | 签到/退窗口偏移 |
| 班次归属 | 直接设置/部门班次/无班次 | 班次匹配逻辑 |
| 缺勤 | 缺勤/正常 | 缺勤表过滤 |

### 考勤类型（Attendance Type）
| 值 | 来源 | 说明 |
|----|------|------|
| 正常 | 考勤机/导入/补卡 | 默认类型 |
| 出差 | 补卡 | 出差补卡 |
| 回工 | 系统 | 回工登记自动生成 |
| 病假/事假/特殊假/年假 + 6拓展 | 请假登记 | 请假类型 |
| 回工 | 回工 | 回工类型 |

### 考勤来源（Attendance Source）
| 值 | 说明 | 删除权限 |
|----|------|---------|
| 考勤机 | 考勤机同步 | 可删除 |
| 补卡 | 手动补卡 | 可删除 |
| 导入 | Excel/TXT导入 | 可删除 |
| 系统 | 回工/请假自动生成 | 不可删除 |

### 补卡类型（Reissue Type）
| 值 | 说明 |
|----|------|
| 正常 | 默认 |
| 出差 | 出差补卡 |

### 状态（Status）
| 实体 | 状态值 | 流转 |
|------|--------|------|
| 回工登记 | 草稿、生效 | 草稿↔生效 |
| 请假登记 | 草稿、审批中、生效、关闭、取消 | 草稿→生效↔草稿 |

### 考勤异常（Attendance Anomaly）
| 值 | 条件 |
|----|------|
| 正常 | 有上下班打卡 或 有请假 |
| 没上班 | 四班次时间全空 + 未请假 |
| 缺卡 | 非没上班 + (某班次缺上班或缺下班) + 未请假 |

### 日期类型（Date Type）
| 值 | 判定条件 |
|----|---------|
| 工作日 | 非周末非节假日 |
| 周末 | 周六/周日 且 未在节假日设为工作日 |
| 节假日 | 在节假日设置日期范围内 |

### 回工时长（Return Work Duration）
| 值 | 开始时间 | 结束时间 |
|----|---------|---------|
| 全天 | 默认早班上班时间 | 默认午班下班时间 |
| 上午 | 默认早班上班时间 | 默认早班下班时间 |
| 下午 | 默认午班上班时间 | 默认午班下班时间 |

## 6. Edge Cases & Constraints

### 6.1 时间边界
- 考勤日期范围最大3个月跨度
- 请假登记查询最大1年跨度
- 月考勤统计查询最大3年跨度
- 日考勤计算起始/结束日期不可>今天或<3个月前
- 600+员工月考勤明细表打印限制

### 6.2 数据边界
- 工号长度: 20字符
- 事由长度: 200字符
- 考勤机名称长度: 20字符
- 小组出勤表最多显示400个小组
- 员工选择界面: 6x10网格

### 6.3 计算边界
- 加班时数按0.5小时四舍五入
- 出勤时数 = 正班时数 + 带薪假时数
- 工作时数 = 正班时数 + 工作日加班 + 周末加班 + 节假日加班
- 迟到 = 早班迟到 + 午班迟到
- 迟到早退 = 正班迟到早退 + 工作日加班迟到早退 + 周末加班迟到早退 + 节假日加班迟到早退

### 6.4 状态边界
- 回工登记: 仅草稿可编辑/删除/生效；仅生效可反生效
- 请假登记: 仅草稿可编辑/删除/生效；仅生效可反生效
- 考勤记录: 来源=系统不可删除
- 日考勤明细: 来源=系统不可删除

### 6.5 特殊人群
- 孕期员工: 早/午班签到/退时间窗口偏移
- 哺乳期员工: 早/午班签到/退时间窗口偏移
- 离职员工: 仅近3月内离职的员工可见
- 产假员工: 月考勤明细表需过滤

### 6.6 并发与性能
- 日考勤计算运算量大，需进度条
- 考勤同步需展示进度条
- TXT/Excel导入需进度条
- 同步间隔分钟数：避免短时间重复更新
