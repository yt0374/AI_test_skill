# Test Data Inventory: 考勤模块

## 1. User/Role Data

| Category | Variation | Expected Behavior |
|----------|-----------|-------------------|
| 普通员工 | 有班次、在职 | 正常考勤登记、请假、日考勤统计 |
| 孕期员工 | 孕期标记=是 | 使用孕期上下班时间(晚于标准时间)进行考勤计算 |
| 哺乳期员工 | 哺乳期标记=是 | 使用哺乳期上下班时间进行考勤计算 |
| 离职员工(近3月) | 离职日期在近3月内 | 考勤登记/日考勤统计下拉可选 |
| 已离职员工(>3月) | 离职日期超过3月 | 不在下拉列表中 |
| 无班次员工 | 未直接设置班次 | 按部门+日期获取最新班次 |
| gm.人事#角色 | 不同权限层级 | 操作权限控制(待明确) |

## 2. Shift Data

| Category | Variation | Test Data |
|----------|-----------|-----------|
| 一班制 | 仅有早班 | 早班标准时间 + 早班签到/签退时间窗 |
| 二班制 | 早班+午班 | 早/午班各标准时间 + 时间窗 |
| 三班制 | 早班+午班+晚班 | 全部班次时间 + 晚班连续自动考勤=true |
| 孕期特殊时间 | 早班孕期上班时间/下班时间 | 晚于标准上班、早于标准下班 |
| 哺乳期特殊时间 | 早班哺乳期上班时间/下班时间 | 同上 |

## 3. Attendance Type Data

| Type | Source | Usage |
|------|--------|-------|
| 正常 | 考勤机/导入/补卡 | 普通打卡记录 |
| 出差 | 补卡 | 出差补卡 |
| 回工 | 系统(回工登记自动生成) | 回工生成的考勤记录 |
| 病假 | — | 日考勤中统计 |
| 事假 | — | 日考勤中统计 |
| 特殊假 | — | 日考勤中统计 |
| 年假 | — | 日考勤中统计(计薪) |
| 拓展假类1~6 | — | 默认不启用，按客户配置 |

## 4. Date Type Data

| Variation | Condition | Impact |
|-----------|-----------|--------|
| 工作日 | 非周末、非节假日 | 计算正班时数、迟到早退 |
| 周末 | 星期六/日 且 未设为工作日 | 计算周末加班时数(字体绿色) |
| 节假日 | 节假日设置范围内 | 计算节假日加班时数(字体红色) |
| 法定调休 | 节假日设为工作日 | 按工作日计算 |

## 5. Attendance Exception Data

| Variation | Condition | Display |
|-----------|-----------|---------|
| 正常 | 有上下班打卡 或 周末/节假日 | 正常显示 |
| 没上班 | 工作日：4个时段全空 且 未请假 | 红色字体 |
| 缺卡 | 工作日：部分时段缺失 且 未请假 | 红色字体 |

## 6. Entity ID Conventions

| Entity | Pattern | Notes |
|--------|---------|-------|
| 工号 | 长度20 | 员工档案关联 |
| 考勤机 | 长度20 | 考勤机设备编号 |
| 设备登录号 | Badgenumber | 考勤机USERINFO关联字段 |
| 回工登记ID | UUID/自增 | 回工考勤记录生成入参 |
| 请假ID | UUID/自增 | 日请假生成入参 |
