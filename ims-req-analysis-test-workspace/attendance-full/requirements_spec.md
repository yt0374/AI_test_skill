# Requirements Specification: IMS 考勤模块

## 1. Overview + Business Context

IMS 人事考勤子系统，为制造型企业提供完整的考勤管理能力。涵盖考勤数据采集（考勤机同步/导入/补卡）、回工登记、请假登记、日/月考勤统计、小组/员工出勤报表。

**更新记录**: 2025-12-12 初始设计 → 2026-03-11 现场变更 → 2026-04-29 月考勤明细表过滤产假员工

## 2. Functional Points

### 2.1 考勤登记 (Attendance Registration)
| ID | 功能点 | 说明 |
|----|--------|------|
| F001 | 考勤记录查询 | 按日期/组别/员工/类型/部门/来源多维查询，日期跨度≤3月 |
| F002 | 考勤同步 | 从考勤机数据库(Access)同步打卡数据，类型=正常，来源=考勤机 |
| F003 | Excel导入 | 选择Excel批量导入考勤记录，类型=正常，来源=导入 |
| F004 | txt导入 | 选择txt批量导入考勤记录，类型=正常，来源=导入 |
| F005 | 补卡 | 按工号+日期手动补卡，生成类型=正常/出差，来源=补卡的考勤记录 |
| F006 | 删除考勤记录 | 单条/批量删除，来源=系统不可删除，删除后自动重算日考勤统计 |
| F007 | 导出 | 导出当前查询结果为Excel |

## 3. Business Rules

| ID | 规则 | 来源 |
|----|------|------|
| R001 | 考勤登记来源=系统不可删除 | 考勤登记.删除 |
| R002 | 批量删除只能删除来源≠系统的记录 | 考勤登记.批量删除 |
| R003 | 删除考勤记录后自动重算对应工号+日期的日考勤统计 | 考勤登记.删除/批量删除 |
| R004 | 回工登记草稿状态才能编辑/删除 | 回工登记.交互 |
| R005 | 回工登记员工数=0时不能生效，告警"没有登记员工，不能生效。" | 回工登记.生效 |
| R006 | 回工登记反生效：回工日期<今天时confirm"回工日期小于今日，反生效会影响已有考勤统计，是否继续？" | 回工登记.反生效 |
| R007 | 请假登记草稿状态才能编辑/删除 | 请假登记.交互 |
| R008 | 请假登记反生效：开始时间<今天时confirm | 请假登记.反生效 |
| R009 | 请假登记保存时校验时间重叠："请假时间范围内存在其他请假。" | 请假登记.编辑.确认 |
| R010 | 日考勤补卡禁用条件：已打卡且时间合法 或 存在对应班次请假时数>0 | 日考勤统计.补卡按钮 |
| R011 | 日考勤统计日期跨度≤3月，起始日期不可>今天或<3月前 | 日考勤计算 |
| R012 | 批量补卡需选中相同日期+相同班次 | 批量补卡 |
| R013 | 月考勤统计只能查近3年生效数据 | 月考勤统计.查询 |
| R014 | 月考勤明细表打印员工人数超过500时提醒 | 月考勤明细表 |
| R015 | 小组出勤表最多400个小组，不分页 | 小组出勤表 |
| R016 | 考勤日期查询跨度≤3月，请假登记查近1年 | 各查询条件 |

## 4. Data Flow

```
考勤机(Access DB) ──[考勤同步]──→ 考勤登记 ──→ 日考勤统计 ──→ 月考勤统计
                                      ↑                ↑
Excel/TXT导入 ────────────────────────┘                │
补卡 ──────────────────────────────────┘                │
回工登记 ──[生效]──→ 回工考勤记录 ──→ 日考勤统计 ──────┘
请假登记 ──[生效]──→ 日请假明细 ──→ 日考勤统计 ──────┘
                                      │
小组出勤表 ←── 日考勤统计汇总(按日期/部门/组别)
员工缺勤表 ←── 日考勤统计汇总(按工号缺勤>0)
```

**跨模块依赖**: 人事模块(员工档案→工号/部门/组别/班次/孕期/哺乳期)、系统模块(节假日设置、假类设置、班次设置、系统选项)

## 5. Cross-Module Dependencies

| 依赖模块 | 依赖数据 | 用途 |
|----------|---------|------|
| 人事-员工档案 | 工号、姓名、部门、组别、班次、孕期、哺乳期、设备登录号 | 考勤关联、补卡校验、日考勤生成 |
| 人事-班次设置 | 早/午/晚班标准时间、签到/签退时间窗、工时比例 | 日考勤计算核心参数 |
| 人事-假类设置 | 假类名称、计薪比例 | 请假登记、带薪假时数计算 |
| 系统-节假日设置 | 节假日日期、法定调休 | 日期类型判定、日考勤统计 |
| 系统-系统选项 | 考勤类型、假类拓展、请假自动删除签到时间 | 考勤配置 |
| 考勤机(Access) | CHECKINOUT表+USERINFO表 | 考勤机打卡数据同步 |

## 6. Data Entities & Variations

### 核心状态机

**请假登记状态**:
```
草稿 ──[生效]──→ 生效 ──[反生效]──→ 草稿
草稿 ──[删除]──→ (删除)
```
> 注: 请假登记只有草稿和生效两种状态，没有取消状态。功能点图中取消为误导。请假不生成考勤登记记录。考勤计算完成后才执行请假删除签到。

**回工登记状态**:
```
草稿 ──[生效]──→ 生效 ──[反生效]──→ 草稿
草稿 ──[删除]──→ (删除)
```

### 考勤类型枚举
正常、出差、回工、病假、事假、特殊假、年假 + 6个拓展假类(默认不启用)

### 考勤来源枚举
考勤机、补卡、导入、系统

### 考勤异常枚举
正常、没上班、缺卡

### 日期类型枚举
工作日、周末、节假日

## 7. Edge Cases & Constraints

| 场景 | 约束 |
|------|------|
| 考勤日期查询 | 跨度≤3月 |
| 请假登记查询 | 近1年 |
| 月考勤统计查询 | 近3年 |
| 日考勤计算起始日期 | 不可>今天，不可<3月前 |
| TXT/Excel导入 | 需进度条，大数据量性能优化 |
| 考勤同步 | 需进度条展示，记录同步完成时间避免重复读取 |
| 回工登记员工选择 | 一屏显示6×10=60个员工 |
| 批量补卡 | 必须同日期+同班次 |
| 月统计打印 | 员工>500人提醒并停止 |
| 小组出勤表 | 最多400小组，不分页 |
| 晚班连续自动考勤 | 午班有签到→晚班上班=晚班标准上班时间 |
| 加班时数 | 按0.5小时四舍五入(1.149→1H, 1.15→1.5H) |
| 考勤起扣时长 | 系统配置项，迟到早退超过此时长才扣 |
